
## Cómo ejecutar el proyecto

Para correr el proyecto, abre dos terminales y ejecuta lo siguiente:

Terminal 1 (Frontend)
cd frontend
npm install
npm run dev

Terminal 2 (Backend)
cd backend
npm install
npm run dev
